# Test Results

This folder contains JSON output from various test runs and search queries.

## Result Files

### Smoke Test Results
- **smoke-test-v3-results.json** - Results from comprehensive Search V3 smoke tests
- **smoke-test-active-products-results.json** - Results from active products smoke tests

### Search Query Results
- **search-results.json** - Default output file for `test-search-cli.mjs`
- **casual-friday-test.json** - "casual friday" contextual query results
- **gym-workout.json** - "gym workout" contextual query results
- **friday.json** - "friday" query results
- **tennis.json** - "tennis" query results
- **leather-shoe.json** - "leather shoe" product search results

## File Format

All result files follow a consistent JSON structure:

```json
{
  "query": "search query",
  "k": 24,
  "server_time_ms": 420,
  "total_time_ms": 850,
  "results": [
    {
      "id": "product-uuid",
      "name": "Product Name",
      "brand": "Brand Name",
      "price": "$100",
      "type": "Pants",
      "gender": "female",
      "score": 0.0325,
      "video_url": "https://...",
      "description": "..."
    }
  ],
  "summary": {
    "total": 10,
    "with_video": 9,
    "avg_score": 0.0298,
    "score_range": [0.0285, 0.0325]
  }
}
```

## Usage

Results can be analyzed using:
- `jq` for command-line JSON processing
- JSON viewers/formatters
- Import into data analysis tools

Example:
```bash
# View top 3 results from a test
jq '.results[:3] | .[] | {name, brand, score}' test-results/casual-friday-test.json

# Count products with video
jq '.summary.with_video' test-results/search-results.json

# Extract all scores
jq '.results[].score' test-results/gym-workout.json
```

## Cleanup

These files can be safely deleted and regenerated by running the corresponding test scripts.
